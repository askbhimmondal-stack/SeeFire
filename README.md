The Fire Alarm System is a safety monitoring application designed to detect potential fire hazards and alert users in real time. The system continuously monitors environmental conditions such as temperature, smoke levels, or fire signals and triggers an alarm when dangerous thresholds are detected.

The main purpose of this project is to provide early fire detection and warning, helping to protect lives, buildings, and valuable assets.
